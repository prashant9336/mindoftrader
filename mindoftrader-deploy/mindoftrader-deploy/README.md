# MindOfTrader

Behavioral Intelligence System for Indian Options Traders.

> "Your charts show the market. We show your mind."

## Deploy

This is a static single-page app. No build step needed.

### Via Vercel CLI
```bash
npm i -g vercel
vercel --prod
```

### Via GitHub + Vercel Dashboard
1. Push this folder to a GitHub repo
2. Import repo on vercel.com
3. Deploy — done
